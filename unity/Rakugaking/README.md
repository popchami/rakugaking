# Rakugaking Unity Project

## Status

Unity project initialization pending.

This folder is reserved for the Unity project.

```text
rakugaking/
└── unity/
    └── Rakugaking/
```

## Important Note

A real Unity project must be created by Unity Editor / Unity Hub / Unity command line.

GitHub file creation alone cannot fully generate and verify a Unity project.

## Recommended Setup

Use this folder as the Unity project location:

```text
unity/Rakugaking/
```

Recommended project type:

- 3D project template
- Stable Unity LTS version available in the development environment

## First Unity Goal

The first goal is not gameplay.

The first goal is only:

- Unity opens the project
- Play mode starts
- No red Console errors appear
- A simple empty scene is available

## Do Not Implement Yet

Do not implement these yet:

- Battle system
- Drawing system
- Character programming system
- UI system
- Save system

## Owner Check

After the Unity project is created, the owner should only check:

- Can Unity open the project?
- Can Play mode start?
- Are there red Console errors?

## Next Step

Create the Unity project in this folder using Unity Editor, Unity Hub, Codex, or another development environment that can run Unity.
