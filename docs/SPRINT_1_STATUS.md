# Sprint 1 Status

## Sprint Name

Minimum Battle Prototype

## Phase

Implementation

## Status

Started / Initial implementation added / Shrinking ring rule integrated

## Implemented Scope

Sprint 1 is limited to:

- Unity project minimum files
- PrototypeBattle scene builder
- flat arena generation
- placeholder humanoid fighter generation
- Rigidbody / Collider / ConfigurableJoint setup
- BattleController
- HP system
- timer
- shrinking ring rule
- emergency fall detection
- simple collision damage
- simple win/loss/draw result
- minimal OnGUI debug HUD

## Explicitly Not Implemented

- drawing feature
- drawing-to-physics conversion
- command system
- online battle
- monetization
- multiple character species
- full UI

## How to Generate PrototypeBattle.unity

Open the repository as a Unity project, then run:

```txt
Unity Menu > Rakugaking > Sprint 1 > Build Prototype Battle Scene
```

This creates:

```txt
Assets/Rakugaking/Scenes/PrototypeBattle.unity
Assets/Rakugaking/Prefabs/Character/PlaceholderFighter.prefab
Assets/Rakugaking/Prefabs/Battle/Arena.prefab
```

## How to Verify

1. Open `PrototypeBattle.unity`.
2. Press Play.
3. Confirm two placeholder fighters spawn.
4. Confirm the fighters move automatically.
5. Confirm HP is displayed.
6. Confirm the timer counts down.
7. Confirm the shrinking ring rule applies outside-ring HP damage.
8. Confirm falling far below the arena ends the battle by emergency fall.
9. Confirm a result is displayed.

## Notes

The current auto movement is not the final command system.

It exists only so Sprint 1 can verify that the battle loop works before command programming is implemented.

Ring-out is no longer the main battle rule. The main arena pressure rule is a shrinking ring.

## Known Limitations

- The scene and prefabs are generated by an Editor menu item, not committed as `.unity` / `.prefab` files yet.
- Physics tuning is rough.
- Collision damage is simple and may need tuning.
- Shrinking ring visual feedback is still minimal.
- The HUD uses OnGUI for quick verification only.
