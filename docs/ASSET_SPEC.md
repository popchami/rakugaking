# Asset Spec

## Purpose

Define the minimum art assets required for the first playable MVP.

## Art Direction

- Simple
- Readable
- Funny
- Bright and approachable

The prototype should prioritize clarity over visual quality.

## Environment Assets

### Arena
- Flat platform
- Simple background
- Ring-out area

## Character Assets

Prototype:
- Placeholder humanoid body
- Simple capsules/boxes are acceptable

MVP:
- Head
- Body
- Upper arms
- Lower arms
- Hands
- Upper legs
- Lower legs
- Feet

These are generated from the player's drawings.

## UI Assets

- Logo
- Buttons
- HP bar
- Timer
- Command icons
- Result icons

## Icons

Commands:
- Punch
- Swing
- Guard
- Kick
- Jump
- Crouch

## Audio (Future)

- Punch hit
- Kick hit
- Fall
- Victory
- Defeat
- UI click

## VFX (Future)

- Hit spark
- Dust
- Ring-out effect
- Victory effect

## Asset Priority

Phase 1:
1. Arena
2. Placeholder fighter
3. Basic UI

Phase 2:
1. Drawing-generated body parts
2. Command icons
3. Improved effects
