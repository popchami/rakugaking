## Project Summary

rakugaking is a Unity game where players draw 3D characters by parts, assign simple actions, and make them fight automatically.

The goal is not perfect control. The goal is chaotic, funny, physics-based battles where the drawing itself creates strategy.

## Core Instructions for AI

- Separate Confirmed, Assumption, and Unknown.
- Do not start coding until the phase allows implementation.
- Keep programming simple for the player.
- Prioritize the MVP: battle using a drawn character.
- Preserve online battle, character sharing, and monetization as important product goals.
