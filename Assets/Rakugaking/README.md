# Rakugaking Unity Assets

This folder contains the Unity implementation for rakugaking.

## Sprint 1 Scope

Sprint 1 is limited to the minimum battle prototype:

- PrototypeBattle scene generation
- flat arena
- placeholder humanoid fighters
- Rigidbody / Collider / Joint setup
- BattleController
- HP system
- timer
- ring-out detection
- simple win/loss display

Do not implement yet:

- drawing features
- drawing-to-physics conversion
- command system
- online battle
- monetization
- multiple character species
- full UI
